# 📊 Super Sales Dashboard

An interactive **Sales Analysis Dashboard** built using **Power BI** and **Microsoft Excel** to analyze sales performance, profit, customer segments, shipping modes, and regional trends.

## 📌 Project Overview

This dashboard helps businesses monitor key sales metrics and identify trends for better decision-making.

## 🛠️ Tools Used

- Microsoft Excel
- Power BI

## 📈 Dashboard Features

- Total Sales
- Total Profit
- Total Quantity
- Average Delivery Time
- Sales by Region
- Sales by Category
- Sales by Sub-Category
- Sales by Segment
- Sales by Payment Mode
- Sales by Ship Mode
- Monthly Sales Trend
- Monthly Profit Trend

## 💡 Key Insights

- Consumer segment contributes the highest sales.
- Office Supplies is the top-performing category.
- Standard Class is the most-used shipping mode.
- COD is the most common payment method.
- Sales show seasonal variation across months.
